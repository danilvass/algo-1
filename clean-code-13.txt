1) Время связывания: при загрузке приложения.
class ConfigurationController

Константы appVersion, backendUrl, defaultRegionId и т д. При загрузке приложения загружаем из конфигурационного файла настройки, которые не меняются за время жизни приложения.

2) Время связывания: отрисовка экрана.

let bgColor: UIColor = Colors.defaultBgColor()

При отрисовке экранов считываются различные данные, которые зависят от выбранных пользователем настроек (светлая/темная тема, размер шрифта и так далее). 

3) Время связывания: вычисление значения при обращении.

var remainingTime: Double {
	return appPreferences.lastUpdateDate - Date() 
}

В Swift есть computed properties, значения которых вычисляются каждый раз при обращении к ним. Полезно использовать для вычисления динамичных значений (оставшееся время, текущий тип пользователя).
